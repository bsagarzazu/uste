# Página web estática de USTE

Sitio web oficial de USTE, una solución web moderna, bilingüe y de alto rendimiento diseñada para una empresa de mensajería líder en Vitoria-Gasteiz con más de 30 años de trayectoria.

## Características principales

- **Bilingüe (ES/EU)**: Soporte completo para Castellano y Euskera con selector de idioma integrado.
- **Mobile-First**: Diseño totalmente responsivo y optimizado para dispositivos móviles, incluyendo botones de acción rápida.
- **SEO Local Optimizado**: Configuración avanzada de metadatos, Open Graph para redes sociales y **JSON-LD** para mejorar el posicionamiento en Vitoria-Gasteiz.
- **Alto Rendimiento**: Carga ultrarrápida gracias a Vite y optimización de assets.
- **Accesibilidad**: Navegación semántica y atributos ARIA para una mejor experiencia de usuario.

La URL del sitio es: [https://bsagarzazu.github.io/uste/](https://bsagarzazu.github.io/uste/)
